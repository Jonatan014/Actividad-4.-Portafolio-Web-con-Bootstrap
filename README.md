# Descripción del proyecto

Este proyecto consiste en el desarrollo de un portafolio web profesional utilizando **Bootstrap 5** como framework CSS, tomando como base la plantilla **DevFolio** de BootstrapMade.

El objetivo fue personalizar completamente la plantilla para crear un sitio web que representara mi portafolio, mostrando mis habilidades, proyectos académicos y medios de contacto.

Framework utilizado:

- Bootstrap 5.3
- HTML5
- CSS3
- JavaScript

Plantilla utilizada:

DevFolio - BootstrapMade

https://bootstrapmade.com/devfolio-bootstrap-portfolio-html-template/

# Proceso de creación

Para elaborar este proyecto se siguieron los siguientes pasos:

1. Se investigaron diferentes plantillas Bootstrap.

2. Se seleccionó la plantilla DevFolio por su diseño profesional.

3. Se descargó la versión gratuita desde BootstrapMade.

4. Se abrió el proyecto en Visual Studio Code.

5. Se personalizó toda la información del portafolio.

6. Se reemplazaron los textos de ejemplo por información personal.

7. Se cambió la fotografía principal por una fotografía profesional.

8. Se modificaron las habilidades mostradas.

9. Se actualizaron las áreas de interés.

10. Se sustituyeron los proyectos originales por proyectos académicos desarrollados durante la carrera.

11. Se reemplazaron las imágenes de ejemplo por capturas de pantalla de cada proyecto.

12. Se modificó la información de contacto.

13. Se realizaron pruebas para verificar que todas las secciones funcionaran correctamente.

14. Finalmente se publicó el proyecto utilizando GitHub Pages.
